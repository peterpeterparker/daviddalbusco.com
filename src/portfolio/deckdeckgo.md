# DeckDeckGo

> This project is deprecated.

DeckDeckGo was an open-source alternative to PowerPoint and Google Slides which I built with web technologies. It ran in the browser on any device, no installation required, and let anyone create, present, and share content.

![A screenshot of the DeckDeckGo editor](/images/portfolio/screenshots/deckdeckgo.webp)

## Fact sheet

**Website:** [DeckDeckGo](https://deckdeckgo.com/)

**Technology:** Mostly [StencilJS](https://stenciljs.com/) and lots of TypeScript/JavaScript.

**Infrastructure:** [Google Cloud](https://firebase.google.com/) and [Amazon AWS](https://aws.amazon.com/) service.

## Open source

This project is open source and available on [GitHub](https://github.com/deckgo/deckdeckgo)
