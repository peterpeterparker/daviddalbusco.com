So much snow!

My original idea was to run the Via Alpina Stage 7 from Altdorf (UR) to Engelberg, but it felt too risky to continue the route and pass the Surenenpass.

It was still an amazing journey, with stunning views and very few other hikers around.

However, if you plan to follow this route, I advise you to check out the official website for the trail:

- https://schweizmobil.ch/en/hiking-in-switzerland/route-1/stage-7
